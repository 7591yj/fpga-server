FPGA.S

Welcome

This is the onboard page of FPGA.S, an integrated FPGA job management interface.
Use this panel to monitor device activity, manage job submissions, and review results from completed tasks.

Navigation

  • Stat: Inspect the current FPGA device status, active workloads, and queued jobs in real time. Useful for verifying resource availability before adding new tasks.
  • Queue: Configure and submit new jobs. Specify device targets, bitstreams, and runtime parameters. Queue entries will be processed sequentially(FIFO) according to availability.
  • Result: Retrieve logs and output data for completed jobs. Each record includes timestamps, resource metrics, and compilation or deployment summaries.

Instructions

Select your device at the right top of the navbar. If there is no device listed, or if the desired device is not present, please click the 'refresh' button next to it. If you still do not see the device even after refreshing, please contact the administrator through here.

After selecting the device, all menus will be shown according to the device. Note that only the information for the selected device is shown.

If you wish to check ALL the job result no matter the device, please go ahead and click the 'show all' button on the right top side of the result page. Please do note that for other pages except for the result page, showing all devices are NOT SUPPORTED by design.

Notes

This app uses Digilent Adept for the actual programming / running jobs.

The source code is availiable in the GitHub repository.
